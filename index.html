Intro

This SilverStripe modules is meant to give a site owner a brief overlook at how well the backend is feeling.

With this module you get three new reports in the admin > reports in silverstripe

Installation:

  1. Download or clone the module into the root document folder of your SilverStripe project.
  2. Decompress and extract the module.
  3. Run the "dev/build".
  4. Profit.

Command line installation

wget https://github.com/downloads/amoebas/silverstripe-serverstatus/serverstatus.tar.gz
tar xvf serverstatus.tar.gz
./sapphire/sake dev/build

Usage

Log in to the admin and go to the report section.
